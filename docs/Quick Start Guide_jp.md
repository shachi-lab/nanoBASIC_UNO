# 📘nanoBASIC UNO - Quick Start Guide（日本語版）
このガイドでは、nanoBASIC UNO を **最短で動かすための手順** をまとめています。  
まずは一度動かしてみて、BASICインタプリタとしての雰囲気を掴んでください。

---

## 1. 必要なもの
- Arduino UNO（ATmega328P）
- Arduino IDE（最新版）
- USBケーブル
- シリアルモニタ（Arduino IDE付属でOK）

---

## 2. nanoBASIC UNO の書き込み

1. `nanoBASIC_UNO.ino` を Arduino IDE で開く  
2. ボードを **Arduino UNO** に設定  
3. シリアル速度を **115200bps** に設定（後で使う）  
4. **書き込み（Upload）** を実行

書き込み完了後、シリアルモニタ（115200bps）を開くと、次のような表示が出ます：

```
nanoBASIC UNO Ver 0.14
OK
```
> ⚠️シリアルモニタの「改行コード」は **CR/LF** に設定してください。
  （設定していない場合、コマンドが正しく送信されないことがあります）

---

## 3. まずは簡単なコマンドを試す

### 文字列を表示
```
PRINT "HELLO"
```

または短縮形：
```
? "HELLO"
```

### 変数を使ってみる
```
A = 10
PRINT A
```

### 四則演算
```
PRINT 1 + 2 * 3
```
### 複数のステートメントを1行で実行する
```
A=1:B=2:C=(A+B)<<3:PRINT "C="C
```
```
FOR I=0 TO 10:PRINT I*I:NEXT
```
---

## 4. 最初のプログラムを作る（LED を点滅させる）

### プログラムモードに入る
```
PROG
```

### 以下の行をそのまま入力する
```
> DO
> OUTP 13, 1
> DELAY 500
> OUTP 13, 0
> DELAY 500
> LOOP
> #
```
`#` を入力すると PROG モードを終了します。

---

## 5. プログラムを実行する
```
RUN
```

Arduino UNO の **13番LED（Lピン）** が点滅すれば成功です ✨

---

## 6. 実行を中断する（ブレーク）
シリアルモニタから **Ctrl-C** を送信すると中断できます。

> ⚠️nanoBASIC UNO の実行中断（ブレーク）は Ctrl-C（0x03）のみ対応しています。
これは BASIC インタプリタの挙動を明確にするための仕様です。
Arduino IDE のシリアルモニタでは Ctrl-C を送信できないため、
中断を使用する場合は Tera Term などの外部ターミナルをご利用ください。


中断した場所から再開：
```
RESUME
```

---

## 7. プログラムの内容を見る
```
LIST
```

---

## 8. プログラムを消去する
```
NEW
```

---

## 9. もっと試してみよう

### 数値の入力
```
INPUT A
PRINT A
```

### アナログ入力
```
PRINT ADC(0)
```

### 乱数
```
RANDOMIZE 1
PRINT RND(100)
```
